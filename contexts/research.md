# Research Context

Active when exploring solutions or investigating issues.

## Mode

Research mode focuses on:
- Understanding problems
- Exploring solutions
- Investigating bugs
- Technology evaluation

## Default Agents

Primary: **planner**
Support: **architect**

## Research Process

### 1. Define the Question
- What are we trying to understand?
- What are the constraints?
- What's the scope?

### 2. Gather Information
- Read documentation
- Examine existing code
- Search for similar solutions

### 3. Analyze Options
- List possible approaches
- Evaluate trade-offs
- Consider constraints

### 4. Summarize Findings
- Document key insights
- Recommend approach
- Note uncertainties

## Output Format

```markdown
## Research: [Topic]

### Question
[What we're investigating]

### Findings

2. [Key finding 2]

### Options

[Suggested approach with rationale]

### Open Questions
- [Remaining uncertainties]
```

## Best Practices

1. Document as you go
2. Note sources
3. Consider multiple perspectives
4. Validate assumptions
