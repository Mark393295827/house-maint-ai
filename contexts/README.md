# Project Context

This directory contains context definitions that provide agents with project-specific knowledge.

## Usage

Context files are loaded by agents to understand:
- Project structure and conventions
- Technology stack details
- Business domain knowledge
- Team preferences and standards

## File Naming Convention

- `project.md` - General project overview
- `tech-stack.md` - Technology stack details
- `conventions.md` - Coding conventions
- `domain.md` - Business domain context
