# 📱 手机端访问指南

本应用采用 "Mobile-First" 设计，在手机上访问体验最佳。

## 1. 准备工作
- 确保您的手机和电脑连接到**同一个 Wi-Fi 网络**。
- 获取电脑的局域网 IP 地址 (例如: `192.168.1.5`)。

## 2. 启动服务
在电脑终端运行：
```bash
npm run dev
```

您会看到如下输出：
```
  ➜  Local:   http://localhost:5173/
  ➜  Network: http://192.168.1.5:5173/  <-- 记下这个地址
```

## 3. 手机访问
1. 打开手机浏览器 (Safari/Chrome)。
2. 输入 `http://<您的IP>:5173` (例如 `http://192.168.1.5:5173`)。
3. 您将看到移动端适配的登录界面。

## 4. 登录操作
1. **输入手机号**: 使用测试账号 `13800138001`
2. **输入密码**: `123456`
3. 点击 **登录** 按钮。
4. 允许相机权限以体验 AI 诊断功能。

## 5. 常见问题
- **无法访问?** 检查电脑防火墙是否允许 Node.js 通信。
- **相机无法打开?** 浏览器可能因为即非 HTTPS 阻止相机。
  - Chrome (Android): 访问 `chrome://flags/#unsafely-treat-insecure-origin-as-secure`，填入IP并启用。
  - Safari (iOS): 在设置中启用 "Web Inspector" 并允许相机访问。
