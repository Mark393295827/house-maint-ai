# House Maint AI 启动指南

本指南将协助您在本地环境启动 House Maint AI 项目。

## 1. 准备工作

请确保您的系统中已安装：
- **Node.js** (推荐 v20 或更高版本)
- **Git**

## 2. 环境配置

项目根目录和 `server` 目录中已经存在 `.env` 文件。请检查以下关键配置：

### 后端 (.env)
- `PORT=3001`: 后端运行端口。
- `JWT_SECRET`: 用于身份验证的密钥。
- `VITE_API_URL`: 前端连接后端的地址（通常为 `http://localhost:3001/api`）。

### 方式 A: 一键启动脚本（推荐）

在项目根目录，双击运行 `run_project.bat` 文件。
该脚本会询问您是否需要安装依赖或运行数据库脚本，并自动在两个新窗口中分别启动前端和后端。

### 方式 B: 手动启动步骤

### 终端 1: 启动后端服务器
1. 进入 `server` 目录：
   ```bash
   cd server
   ```
2. 安装后端依赖（如果尚未安装）：
   ```bash
   npm install
   ```
3. 初始化数据库（如果是第一次运行）：
   ```bash
   npm run init-db
   ```
4. 启动后端开发服务器：
   ```bash
   npm run dev
   ```
   后端通常运行在 `http://localhost:3001`。

### 终端 2: 启动前端服务器
1. 在项目根目录安装前端依赖（如果尚未安装）：
   ```bash
   npm install
   ```
2. 启动前端开发服务器：
   ```bash
   npm run dev
   ```
   前端通常运行在 `http://localhost:5173`。

## 4. 验证运行

- 打开浏览器并访问 `http://localhost:5173`。
- 如果看到前端界面，并且能正常进行登录或浏览（后台无报错），则说明项目已成功运行。

---
> [!TIP]
> 如果在启动过程中遇到端口被占用的错误，请检查是否有其他服务运行在 3001 或 5173 端口。
